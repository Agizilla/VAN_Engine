WHAT IS THIS?
═════════════════════════════════════════════════════════════════════════════

VLC_AudioStudio is an audio player with integrated voice training and cloning:

✓ Play any audio file (MP3, WAV, FLAC, etc)
✓ Train custom voice models from artist samples
✓ Clone voices with 95-99% accuracy
✓ Professional-grade quality


QUICK START (Choose Your Path)
═════════════════════════════════════════════════════════════════════════════

When you open VLC_AudioStudio, you'll see three guide buttons in the toolbar:

  ⚙️  SETUP WIZARD
      └─ New to this? Start here!
      └─ Guides you through installation
      └─ Verifies Python
      └─ Runs setup script

  📖 TRAINING GUIDE  
      └─ Complete technical reference
      └─ Advanced configuration
      └─ Troubleshooting

  ⚡ QUICK START
      └─ Voice training in 5 steps
      └─ How to run training
      └─ How to clone voices


WHAT TO DO FIRST
═════════════════════════════════════════════════════════════════════════════

1. Build the project in Visual Studio
2. Run (F5)
3. Click "⚙️ Setup" button in toolbar
4. Follow the wizard (5 minutes)
5. Click "⚡ Quick Start" button
6. Follow those steps to train your first model


REQUIREMENTS
═════════════════════════════════════════════════════════════════════════════

✓ Python 3.8+ (Install from https://www.python.org/)
✓ 8GB RAM (16GB recommended)
✓ Optional: NVIDIA GPU (10x faster)
✓ 5-10 audio samples (.wav files) of artist voice


THE WIZARDS WILL HANDLE:
═════════════════════════════════════════════════════════════════════════════

✓ Checking Python installation
✓ Running setup script (installs dependencies)
✓ Creating folders
✓ Downloading base models
✓ Guiding audio preparation
✓ Explaining training parameters
✓ Walking through voice cloning
