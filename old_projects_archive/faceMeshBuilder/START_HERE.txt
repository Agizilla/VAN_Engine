# START HERE - Windows Quick Setup

## You Have Python 3.13 Error? Read This! ⚠️

The error you're seeing is a **known compatibility issue** with Python 3.13 and OpenCV.

## 🚀 Fastest Fix (3 Steps)

### Step 1: Double-click this file
```
FIX_INSTALL.bat
```

This will automatically:
- Detect your Python version
- Uninstall incompatible packages
- Install the correct versions
- Test that everything works

### Step 2: Wait for it to complete
You'll see messages showing the installation progress.

### Step 3: Run the application
Double-click:
```
RUN.bat
```

Or in command prompt:
```
python face_avatar_3d.py
```

## ✅ What Gets Fixed

The automatic fix installs these compatible versions:

| Package | Version Required |
|---------|------------------|
| numpy | 1.26.0 or newer |
| opencv-python | 4.10.0 or newer |
| mediapipe | 0.10.14 or newer |
| pillow | 10.0.0 or newer |
| PyOpenGL | 3.1.7 or newer |
| pygame | 2.6.0 or newer |

## 🔧 Alternative: Use Python 3.11 or 3.12

If the automatic fix doesn't work, **using Python 3.11 or 3.12 is recommended**:

1. Download from: https://www.python.org/downloads/
2. Install Python 3.11 or 3.12
3. Open Command Prompt in this folder
4. Run:
   ```
   py -3.11 -m pip install -r requirements.txt
   py -3.11 face_avatar_3d.py
   ```

## 🎓 Technical Details

**Why the error?**
- Python 3.13 is brand new (released October 2024)
- Some packages haven't fully updated yet
- OpenCV needed numpy updates to work with Python 3.13
- Older package versions don't have `_ARRAY_API` attribute

**The fix:**
- Update numpy to 1.26+ (has Python 3.13 support)
- Update OpenCV to 4.10+ (works with new numpy)
- Everything else updates automatically
