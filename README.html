Architecture

SAAS Server — FastAPI server on port 8001 with 71 dynamically-loaded skill hooks. Each hook is a POST endpoint with auto-generated UI forms, JSON tree views, and HTML preview. Routes split across api.py, ui_routes.py, forge.py, midi.py, github_watch.py, and optimiser.py.
Skill System — Skills auto-discovered from saas/src/skills/. Categories: audio (13), video (13), text (7), PRD (8), GitHub, MIDI, Forge, learnings, humor, intent, and more. Each skill declares input/output schema, description, tags, and category.
GitHub Bridge — Versioned agent-repo interaction via gh CLI. Actions: commit, create_pr, list_issues, list_forks, repo_status, create_repo, repo_info, repo_search, repo_read_file. With real-time watchdog polling for new repos.
Prompt Optimiser — POST /api/optimise endpoint with server-side Base64 Algorithm injection. Portal tab for optimising prompts with Goal/Constraints/Steps structure and diff view.
Forge Engine — Multi-agent SSE streaming studio. Spawn N agents with assigned skills, watch them collaborate in real-time through Server-Sent Events. Includes entanglement mode.
MIDI Composer — "Mike" — browser-based MIDI composer with piano keyboard, Web Audio playback, VexFlow sheet music rendering, and JSON export.
Learnings Registry — Shared knowledge JSONL store at ~/.claude/PAI/LEARNINGS/. Agents read before acting, write after learning. CRUD via skill hooks.

Quick Start

# From saas/ directory:
python -m uvicorn server:app --host 127.0.0.1 --port 8001

# Open the portal:
open http://127.0.0.1:8001/hooks/ui/portal

# List all hooks:
curl http://127.0.0.1:8001/hooks/

# Execute a hook:
curl -X POST http://127.0.0.1:8001/hooks/vibe_affirmations \
  -H "Content-Type: application/json" -d '{}'
