# Sprint 7 — State Machine Graphs & Interaction Buffers

## Objective
Give the character persistent internal psychological architecture — behavior trees, encrypted event logs, and adaptive weight interfaces.

## Deliverables
- `behavior_tree.js` — Hierarchical state machine (focus, cadence, engagement, arousal, processing load)
- `interaction_log.js` — Encrypted rolling event buffer (click coords, timing, chapter path)
- `trajectory_adapters.js` — State → IK weight adapters (mood alters posture, expression, idle animation)

## Integration
Behavior tree state feeds Sprint 3's IK weights, Sprint 4's expression targets, and Sprint 6's vocal pitch.
