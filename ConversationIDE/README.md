# Conversation-IDE

Conversation-IDE is a governed, local-first workspace for VAN_Engine.

It combines:
- chat-driven operator workflows
- project file browsing and editing
- build and deployment hooks
- VAN_Engine status, audit, and ISO rule visibility

What makes it different:
- governed actions, not just chat
- offline-first by design
- one workspace brain for chat, files, builds, and audits
- project-aware context instead of a stateless prompt box
- operator mode for inspect, patch, build, and verify flows
